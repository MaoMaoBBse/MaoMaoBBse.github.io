C++17 std::any 的使用方法

2021-10-12- 2021-10-18
笔记-C++std笔记
猫猫白白se

简介

std::any 在 C++17 后加入标准库，主要用于代指任何类型的类型安全容器，用于替代 C 语言的 void*。当你不清楚需要存储的数据是什么类型时，可以采用 std::any 存储。

使用

创建与赋值

std::any 拥有简单易用的拷贝构造和赋值函数，创建与赋值用起来还是很方便的。

std::any a1;       // a1, 类型：无      值：空
std::any a2 = 1;   // a2, 类型：int     值：1
std::any a3{a2};   // a3, 类型：int     值：1
a2 = 1.1;          // a2, 类型：double  值：1.1

获取存储的类型

使用成员函数 type()可以获取存储的数据类型

std::any a1 = 10;
if(a1.type() == typeid(int)){
	std::cout << "a1 = " << std::get<int>(a1) << std::endl;
}

获取存储的值

使用 std::get(std::any)获取，如果类型错误会抛出异常。

std::any a1 = 10;
auto num = std::get<int>(a1); // num: 10
auto& num_r = std::get<int&>(a1); // num_r: 10
num_r = 20; // a1: 10
